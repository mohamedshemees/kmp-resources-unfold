# Kmp Resources Unfold

![Build](https://github.com/mohamedshemees/Kmp-Resources-Unfold/workflows/Build/badge.svg)

<!-- Plugin description -->
Kmp Resources Unfold is a powerful IntelliJ Platform Plugin designed to streamline resource management in Kotlin Multiplatform (KMP) projects. 

Features:
- **Resource Explorer:** Easily browse and filter drawables (XML, PNG, SVG, WebP, JPG) across all your modules.
- **Instant Preview:** View vector and image resources directly within the IDE.
- **Compose Code Generation:** Double-click to open, or single-click to copy the `painterResource` code snippet ready for your Compose Multiplatform code.
- **Module Filtering:** Quickly narrow down resources by specific modules.
<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "Kmp Resources Unfold"</kbd> > <kbd>Install</kbd>

- Manually:
  Download the [latest release](https://github.com/mohamedshemees/Kmp-Resources-Unfold/releases/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
