**Kmp Resources Unfold** is a professional productivity tool designed for **Android Studio** and **IntelliJ IDEA** developers. It bridges a major gap in the IDE by providing visual management for assets across all your Kotlin Multiplatform (KMP) modules.

* **Native XML Rendering**: Render **Android Vector Drawables (.xml)** directly in the list.
* **Unified Asset Explorer**: Browse XML vectors, SVGs, and raster images from all modules.
* **Seamless Integration**: Designed to feel like a native part of the IDE UI.

## 🛠 Features

* **Instant Previews**: High-fidelity rendering of vectors with support for gradients and transparency.
* **Design Tab**: A specialized file editor tab for Vector assets featuring:
    * **Smooth Zoom**: Precise control over preview size.
    * **Reset to 100%**: Native "Actual Size" button for accurate scaling.
    * **Auto-refresh**: Live preview updates as soon as you save file changes.
* **Smart Interaction**:
    * **Single Click**: Copy the resource path to clipboard with an instant notification.
    * **Double Click**: Open the source file directly in the editor.
* **Live Filtering**: Filter assets by category (Vectors/Images) with real-time counts.
* **Smart Localization Support (Strings)**:
    * **Missing Translation Warnings**: Instantly spot strings that are missing translations across your `values` directories.
    * **Quick Navigation**: Use <kbd>Ctrl</kbd> + <kbd>Click</kbd> (or <kbd>Cmd</kbd> + <kbd>Click</kbd>) on a string key to instantly navigate between its translations.

## 🚀 Installation

### Using JetBrains Marketplace (Recommended)
1. Open <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd>
2. Search for `Kmp Resources Unfold`
3. Click <kbd>Install</kbd> and restart your IDE.

Alternatively, go to the [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/31310-kmp-resources-unfold) and install it by clicking the <kbd>Install to ...</kbd> button if your IDE is running.

### Manual Installation
1. Download the latest `.zip` or `.jar` from the [Releases Page](https://github.com/mohamedshemees/Kmp-Resources-Unfold/releases/latest).
2. Open **Android Studio** or **IntelliJ IDEA**.
3. Go to <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
4. Select the downloaded file and restart the IDE.
