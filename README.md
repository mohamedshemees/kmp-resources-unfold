# Kmp Resources Unfold

> [!IMPORTANT]
> **Early Access**: This plugin is currently in the pre-release phase . You can download the latest version and install it manually for now.

**Kmp Resources Unfold** is a professional productivity tool designed for **Android Studio** and **Kotlin Multiplatform** developers. It bridges a major gap in the IDE by providing visual management for assets across all your modules.

## 🌟 Why Kmp Resources Unfold?

While Android Studio is powerful, it natively lacks a quick way to preview multiple assets—especially **Android Vector XMLs**—outside of the editor. This plugin solves that by providing:

* **Native XML Rendering**: Unlike the standard file explorer, this plugin renders **Android Vector Drawables (.xml)** directly in the list, giving you an instant visual overview of your icons.
* **Unified Asset Explorer**: Browse XML vectors, SVGs, and raster images (PNG, JPG, WebP) from all modules in one centralized panel.
* **Seamless Integration**: Designed to feel like a native part of the **Android Studio** UI.

---

## 🛠 Features

* **Instant Previews**: High-fidelity rendering of vectors with support for gradients and transparency.
* **Design Tab**: A specialized file editor tab for Vector assets featuring:
    * **Smooth Zoom**: Precise control over preview size.
    * **Reset to 100%**: Native "Actual Size" button for accurate scaling.
    * **Auto-refresh**: Live preview updates as soon as you save file changes.
* **Smart Interaction**:
    * **Single Click**: Copy the resource path to clipboard with an instant notification.
    * **Double Click**: Open the source file directly in the editor.
* **Live Filtering**: Filter assets by category (Vectors/Images) with real-time counts.

## 🚀 Installation

### Manually (Current Early Access)
Download the latest release from GitHub and install it manually:
1. Download the latest `.zip` or `.jar` from the [latest release](https://github.com/mohamedshemees/Kmp-Resources-Unfold/releases/latest).
2. Open **Android Studio**.
3. Go to <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
4. Select the downloaded file and restart the IDE.

### Using the IDE built-in plugin system (Coming Soon ⏳)
1. Open <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> 
2. Search for `Kmp Resources Unfold`
3. Click <kbd>Install</kbd>

### Using JetBrains Marketplace (Coming Soon ⏳)
Go to [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/MARKETPLACE_ID) and install it by clicking the <kbd>Install to ...</kbd> button in case your IDE is running.
