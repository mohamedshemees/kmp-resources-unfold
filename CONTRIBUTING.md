# Contributing to KMP Resources Unfold

First of all, thank you for showing interest in improving this plugin!

### How can I contribute?

#### 1. Reporting Bugs 🐛
Before creating a new issue, please search the existing issues to see if it has already been reported. If not, use the **Bug Report** template to provide all necessary details.

#### 2. Suggesting Enhancements 💡
We are open to new ideas! Please open an issue first to discuss the feature before spending time on a Pull Request (PR).

#### 3. Pull Requests (PRs) 🛠️
To maintain code quality and project vision:
- **Open an Issue first**: Discuss the change you want to make.
- **Wait for approval**: Don't start coding until we agree on the approach.
- **Follow the style**: Ensure your code follows the existing Kotlin/IntelliJ coding standards.
- **Single concern**: Each PR should address one issue only.

> [!NOTE]
> The maintainer reserves the right to accept or reject any contribution based on the project's roadmap and quality standards.
